# 1-Digit Calculator

Sebuah kalkulator sederhana yang dibuat dengan Next.js untuk melakukan operasi aritmetika dasar pada angka satu digit.

## Gambaran UI 

| 1 | 2 | 3 | AC |
| 4 | 5 | 6 | +  |
| 7 | 8 | 9 | -  |
            | =  |

## Aturan

* **digit1**: digit pertama
* **digit2**: digit kedua
* **operator**: `+` dan `-`


ketika tombol `=` (sama dengan) diklik muncul hasilnya
